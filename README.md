# Enes Weather (No‑Key Version)

A unique, no‑framework weather app that works out of the box using **Open‑Meteo** (no API key).
- City search with geocoding
- Current weather (°C, humidity, wind m/s)
- **3‑day forecast** (min/max) excluding today
- Custom SVG icons, glassmorphism UI

## Run
Just open `index.html` in your browser. Internet is required to call Open‑Meteo APIs.

## Tech
- Geocoding: https://geocoding-api.open-meteo.com/v1/search
- Forecast: https://api.open-meteo.com/v1/forecast
